# npm-run-scripts

[![Greenkeeper badge](https://badges.greenkeeper.io/fisker/npm-run-scripts.svg)](https://greenkeeper.io/)

run npm scripts with a list you can select with arrow key

## install
```sh
npm i -g npm-run-scripts
```

## ustage
```sh
npm-run
```

OR (since v1.0.4)

```sh
npm-run dev
```
